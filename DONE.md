## 📝 ПРИМЕЧАНИЯ

### Валидация формы создания доски

**ДО:**
```javascript
createBoard(title, tripId)
```

**ПОСЛЕ:**
```javascript
createBoard({
  name: "Название",
  description: "Описание",
  tripId: "..."
})
```

Это соответствует требованиям бэкенда, где `name` и `description` обязательны.
