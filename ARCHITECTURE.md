## 🔧 Проверка валидации бэкенда

### Trip (Отпуск)
```
Обязательные поля:
- title: string (название отпуска)
- destination: string (пункт назначения)

Опциональные поля:
- description: string | null (описание)
```

### Board (Доска)
```
Обязательные поля:
- tripId: Guid (ID отпуска)
- name: string (название доски)
- description: string (описание доски)

Опциональные поля:
- Нет
```

## 🛣️ Маршруты приложения

| Маршрут | Компонент | Описание |
|---------|-----------|----------|
| `/` | LoginPage | Страница входа |
| `/register` | RegisterPage | Страница регистрации |
| `/trips` | TripsPage | Список всех отпусков пользователя |
| `/trips/:tripId` | TripDetailsPage | Детали конкретного отпуска |
| `/boards/:boardId` | BoardDetailPage | Детали доски (в разработке) |

## 🎯 Пользовательский интерфейс

### 1. **Страницы аутентификации** (LoginPage, RegisterPage)
- ✅ Красивый дизайн с сине-белой гаммой
- ✅ Карточка в центре экрана
- ✅ Плавная анимация входа
- ✅ Обработка ошибок
- ✅ Кнопки навигации между входом и регистрацией
- ✅ Валидация пароля (минимум 6 символов)

### 2. **TripsPage** (Список отпусков)
- 🏖️ Заглушка "Отпусков еще нет" (красивая иконка ✈️)
- ✅ Сетка карточек отпусков (responsive grid)
- 🔘 Кнопка "+ Создать новый" отпуск
- 🔵 Каждая карточка содержит:
  - Название отпуска
  - Пункт назначения (с иконкой 📍)
  - Описание (если есть)
  - Быстрый просмотр ID
- 📱 Полностью адаптивный дизайн

### 3. **TripDetailsPage** (Детали отпуска)
- ← Кнопка "Вернуться" к списку
- 📋 Полная информация об отпуске:
  - Название
  - Пункт назначения
  - Описание (если есть)
- 👥 Раздел "Участники"
- 📊 Раздел "Доски"
  - Список всех досок этого отпуска
  - Переход на доску при клике
- 📋 Раздел информации с ID отпуска

### 4. **BoardDetailPage** (Детали доски)
- ← Кнопка "Назад"
- 🟦 Пустая доска (заглушка) - в мае будет реализована полная функциональность

## ✨ Ключевые компоненты стилей

### Кнопки
```
.btn-primary    - основная кнопка (синяя)
.btn-secondary  - вторичная кнопка (белая с синей обводкой)
.btn-ghost      - прозрачная кнопка (для второстепенных действий)
.btn-danger     - кнопка опасного действия (красная)
```

### Модальные окна
- `.modal-overlay` - полупрозрачный фон
- `.modal-content` - контент модального окна с анимацией slideUp

### Карточки
- `.trip-card` - карточка отпуска с полосой слева
- `.board-list-item` - элемент доски в списке

### Раскладка
- `.page` - основной контейнер страницы с градиентом
- `.trips-container` - сетка карточек (авто-заполнение, min-width: 300px)
- `.board-layout` - гибкая раскладка для досок (flex)

## 🎓 Примечания по API

### Board API - обновлено
**Функция createBoard теперь принимает объект payload:**
```javascript
// ДО:
createBoard(title, tripId)

// ПОСЛЕ:
createBoard({ name: "Название", description: "Описание", tripId: "..." })
```

Это соответствует валидации бэкенда, где `name` и `description` обязательны.
