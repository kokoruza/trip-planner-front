## 🎯 Компоненты UI

### Кнопки
```
.btn-primary    → Синяя с белым текстом (основные действия)
.btn-secondary  → Белая с синей обводкой (вторичные действия)
.btn-ghost      → Прозрачная (дополнительные действия)
.btn-danger     → Красная (опасные действия)

Hover эффект: Поднятие на 2px, увеличение тени
```

### Карточки
```
.trip-card:
- Синяя полоса слева (4px)
- Белый фон
- Тень при hover
- Плавное поднятие на 4px
- Содержит: название, место, описание, ID

.board-list-item:
- Голубой фон (#dbeafe)
- Синяя обводка (2px)
- Содержит: название, описание
```

### Формы
```
input, textarea, select:
- Белый фон
- Серая обводка (2px)
- При фокусе: синяя обводка + голубая тень
- Padding: 12px 16px
- Border-radius: 8px
- Плавный переход всех свойств
```

### Модальные окна
```
.modal-overlay:
- Полупрозрачный чёрный фон (0.4)
- Абсолютное позиционирование на весь экран

.modal-content:
- Белый фон
- Padding: 32px
- Max-width: 500px
- Анимация slideUp (300ms)
- Тень 0 20px 60px
```

## ⌨️ Навигация

### Основные переходы
```
Login → Register  (ссылка внизу формы)
Login → Trips     (после успешного входа)
Trips → Trip Details (клик по карточке)
Trip Details → Board Detail (клик по доске)
Trip Details → Trips (кнопка "Вернуться")
Board Detail → (назад через браузер или Button)

Выход → Login (везде)
```

---

## 🎭 Анимации

```
1. slideUp (300ms):
   From: opacity 0, transform translateY(20px)
   To:   opacity 1, transform translateY(0)
   
   Применяется к:
   - Auth cards
   - Modal windows

2. Hover effects:
   - Кнопки: +2px elevation
   - Карточки: +4px elevation
   - Links: underline
   - Inputs: border color change

3. Transitions:
   - Duration: 300ms
   - Easing: ease
   - Properties: all
```

## 📋 Валидация формы

### Регистрация
```
✓ Email: валидный email
✓ Password: минимум 6 символов
✓ Confirm Password: совпадает с Password
✓ Все поля обязательны
```

### Создание отпуска
```
✓ Title: не пусто, макс 100 символов
✓ Destination: не пусто, макс 100 символов
✓ Description: опционально, макс 500 символов
```

### Создание доски
```
✓ Name: не пусто, макс 100 символов
✓ Description: не пусто, макс 300 символов
✓ TripId: обязателен (передаётся автоматически)
```

## 📱 Мобильный UI особенности

```
- Padding: 16px вместо 24px
- Font-size: на 2px меньше
- Кнопки на полную ширину в модальных окнах
- Модальные окна: 90vw ширины
- Grid: single column layout
- Touch-friendly: min-height buttons 44px
```
